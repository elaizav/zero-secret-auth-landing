Методологія та інструментальні засоби розробки систем автентифікації за моделлю Zero Secret Server Authentication

Methodology and Instrumental Tools for Developing Authentication Systems Based on the Zero Secret Server Authentication Model

Бакалаврська робота присвячена дослідженню підходів до створення систем автентифікації, у яких секрети користувача не зберігаються на сервері, а критичні механізми захисту реалізуються на клієнтській стороні.

Ключові слова: Zero Secret Server Authentication, системи автентифікації, кібербезпека, клієнтське шифрування, stateless authentication.

Головний опис роботи

Цей лендінг презентує бакалаврську роботу, присвячену дослідженню методології та інструментальних засобів розробки систем автентифікації за моделлю Zero Secret Server Authentication.

Актуальність теми

Сучасні інформаційні системи вимагають високого рівня захисту автентифікаційних даних користувачів. Традиційні підходи передбачають зберігання секретів або похідних від них на сервері, що створює додаткові ризики компрометації. Модель Zero Secret Server Authentication є актуальною, оскільки спрямована на зменшення ризиків витоку даних, підвищення стійкості системи до атак та вдосконалення архітектури безпечної автентифікації.

Мета та завдання дослідження

Мета дослідження

Дослідити методологічні підходи та інструментальні засоби розробки систем автентифікації за моделлю Zero Secret Server Authentication і підготувати основу для створення демонстраційного прототипу.

Основні завдання

  1. Проаналізувати існуючі підходи до реалізації систем автентифікації.
  2. Дослідити принципи моделі Zero Secret Server Authentication.
  3. Визначити інструментальні засоби для реалізації безпечного вебрішення.
  4. Сформувати архітектурний підхід до побудови системи автентифікації.
  5. Підготувати основу для подальшої реалізації демонстраційного прототипу.

Методологія дослідження

У роботі використовується підхід, що поєднує аналіз сучасних моделей автентифікації, принципи безпечної веброзробки, клієнтське шифрування, stateless-підхід до обробки автентифікаційних даних та порівняльний аналіз інструментальних засобів реалізації вебсистем.

Основні методи

  • аналіз наукових і технічних джерел;
  • порівняння архітектурних підходів;
  • моделювання структури вебсистеми;
  • проєктування інтерфейсу односторінкового лендінгу.

Очікувані результати

  • систематизація підходів до побудови безпечних систем автентифікації;
  • обґрунтування доцільності використання моделі Zero Secret Server Authentication;
  • формування структури майбутнього програмного рішення;
  • створення демонстраційного вебпредставлення бакалаврської роботи;
  • підготовка основи для подальшого розширення проєкту.
