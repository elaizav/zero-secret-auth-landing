Огляд бакалаврської роботи

Лендінг презентує тему, основні ідеї, методологію дослідження та очікувані результати бакалаврської роботи.

Про дослідження

Тема роботи

Методологія та інструментальні засоби розробки систем автентифікації за моделлю Zero Secret Server Authentication.

Актуальність

Дослідження присвячене підходам до побудови систем автентифікації, у яких мінімізується залежність від серверного зберігання секретів.

Методологія

Підхід до реалізації

У роботі розглядаються принципи клієнтського шифрування, stateless-підхід до автентифікації та використання сучасних інструментів веброзробки.

Технічні акценти

Ключові напрями

  • безпечне зберігання даних;
  • архітектура без серверних секретів;
  • підтримка сучасних вебтехнологій;
  • зручність і безпечність інтерфейсу.

Очікувані результати

Практичний результат

Створення демонстраційного прототипу лендінгу та технічної основи для подальшого розвитку бакалаврського проєкту.

Дослідницький результат

Формалізація підходів до створення систем автентифікації за моделлю Zero Secret Server Authentication.

Контакти

Контактна інформація буде уточнена на етапі наповнення контентом.
